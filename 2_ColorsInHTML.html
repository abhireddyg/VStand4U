INLINE TAGS IN HTML

1. ALIGN ATTRIBUTES

Text Alignment to left (align ="left")

Text Alignment to Center (align ="center")

Text Alignment to right (align ="right")

2. STYLE ATTRIBUTE

2.1 TEXT & BACKGROUND COLORS

Text Color (style="color: red;")

Background Color (style="background-color: yellow;")


2.2 HTML COLORS

Color using Name

Hex Color (style="color: #FF0000;")

RGB Color (style="color: rgb(255,0,0);")

RGBA Color (style="color: rgba(255,0,0,0.6);")

HSL Color (style="color: hsl(0,100,50);")

HSLa Color (style="color:hsla(0,100,50,0.5);")


3. HEIGHT & WIDTH ATTRIBUTES

Height & Width (style="height: 100px; width: 200px;")
