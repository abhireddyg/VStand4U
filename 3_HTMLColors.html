HTML COLORS

COLORS USING NAMES

Red Color --> (style="color: red;")

Green Color --> (style="color: green;")

Blue Color --> (style="color:blue;")


COLOR USING HEXA DECIMAL

Hex Color (style="color: #FF0000;")

Hex Color (style="color: #00FF00;")

Hex Color (style="color: #0000FF;")


COLOR USING RGB & RGBa

RGB Color (style="color: rgb(255,0,0);")

RGB Color (style="color: rgb(0,255,0);")

RGB Color (style="color: rgb(0,0,255);")

RGBA Color (style="color: rgba(255,0,0,0.6);")

RGBa Color (style="rgba(0,0,0,0.5);"


COLOR USING HSL & HSLa

HSL Color (style="color:hsl(280, 96%, 52%);"

HSL Color (style="color: hsl(0,100%,50%);")

HSLa Color (style="color:hsla(0,100%,50%,0.5);")

HSLa Color (style="color:hsl(280, 96%, 52%,0.4);"
